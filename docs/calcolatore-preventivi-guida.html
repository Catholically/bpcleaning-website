Calcolatore Preventivi

Guida completa per BP Cleaning

Come Funziona

Il calcolatore guida il cliente in 4 semplici step per ottenere una stima di prezzo immediata:

  • Tipo di servizio - Casa, Ufficio, Condominio o Industriale
  • Metratura e frequenza - Slider da 20 a 500 mq + scelta frequenza
  • Localita - Inserimento CAP per verifica copertura
  • Contatti - Nome, telefono, email per ricevere il preventivo
Importante: Il calcolatore fornisce una stima indicativa. Il prezzo finale viene confermato dopo sopralluogo gratuito.

Prezzi Base per Metro Quadro

Tipo Servizio Prezzo/mq Perche questo prezzo
Casa €2.50 Maggiore attenzione ai dettagli, mobili, oggetti personali
Ufficio €2.00 Standard, spazi aperti, meno ostacoli
Condominio €1.80 Spazi comuni ripetitivi (scale, androni)
Industriale €1.50 Grandi superfici, economia di scala

Sconti per Contratto Continuativo

Frequenza Sconto Strategia
Una tantum 0% Prezzo pieno, costi fissi di avvio
Mensile -10% Sconto minimo per contratto base
Quindicinale -15% Buon compromesso qualita/prezzo
Settimanale -25% Massimo sconto per fidelizzazione
Perche sconti cosi alti per settimanale?
Lo sconto del 25% sembra alto ma: fidelizza il cliente a lungo termine, riduce i costi di acquisizione nuovi clienti, e permette di organizzare meglio i turni di lavoro.

La Formula del Calcolo

1. Prezzo Base = Metri Quadri × Prezzo/mq
2. Prezzo Scontato = Prezzo Base × (1 - Sconto)
3. Range = Prezzo Scontato × 0.85 ~ 1.15

Il prezzo viene mostrato come range ±15% per permettere flessibilita in base a:

  • Stato dell'immobile
  • Difficolta di accesso
  • Richieste speciali del cliente
  • Margine di negoziazione commerciale

Esempio Pratico

Scenario: Ufficio 100 mq, pulizia settimanale

Input:

  • Tipo: Ufficio (€2.00/mq)
  • Metratura: 100 mq
  • Frequenza: Settimanale (-25%)

Calcolo:

  • Base: 100 × €2.00 = €200
  • Scontato: €200 × 0.75 = €150
  • Min: €150 × 0.85 = €127
  • Max: €150 × 1.15 = €172
Prezzo mostrato al cliente
€127 - €172

Come Modificare i Parametri

Tutti i parametri sono nel file:

src/components/QuoteCalculator.tsx

Modificare i Prezzi (righe 19-24)

const serviceTypes = [
  { id: "casa", label: "Casa", pricePerSqm: 2.5 },
  { id: "ufficio", label: "Ufficio", pricePerSqm: 2.0 },
  { id: "condominio", label: "Condominio", pricePerSqm: 1.8 },
  { id: "industriale", label: "Industriale", pricePerSqm: 1.5 },
];

Modificare gli Sconti (righe 26-31)

const frequencies = [
  { id: "una_tantum", label: "Una tantum", discount: 0 },
  { id: "settimanale", label: "Settimanale", discount: 0.25 },
  { id: "quindicinale", label: "Quindicinale", discount: 0.15 },
  { id: "mensile", label: "Mensile", discount: 0.10 },
];

Nota: 0.25 significa 25%, 0.10 significa 10%

Modificare il Range ±15% (righe 60-63)

return {
  min: Math.round(discountedPrice * 0.85),
  max: Math.round(discountedPrice * 1.15),
};

Cambia 0.85/1.15 per range piu stretto o largo

Checklist per il Cliente

Domande da verificare:
  • I prezzi €/mq sono realistici per il mercato locale?
  • Lo sconto 25% settimanale e sostenibile economicamente?
  • Il range ±15% da abbastanza margine?
  • Servono altri tipi di servizio? (es. vetri, post-cantiere)
  • Le zone CAP coperte sono corrette?

Zone Coperte (CAP)

Attualmente il calcolatore accetta:

  • 21xxx - Provincia di Varese
  • 20xxx - Provincia di Milano

Per aggiungere altre province (es. Como 22xxx, Monza 20900), modificare la funzione validateCAP nel file.

Riepilogo Strategia Prezzi

Punti di Forza
  • Prezzi competitivi
  • Sconti attraenti per contratti
  • Range protegge i margini
  • Sopralluogo gratuito rassicura
Attenzione
  • Prezzi NON vincolanti
  • Sempre confermare con sopralluogo
  • Valutare caso per caso
  • Aggiornare se costi aumentano
